Código-fonte

"O comando document.getElementById('teste') é escrito em linguagem JavaScript." A fonte utilizada para escrever o código, não é monoespaçada, ou seja, não reserva o mesmo espaço lateral para cada letra. Para arrumar isso, é possível utilizar a tag "code" e deixar a fonte monoespaçada
(fontes monoespaçadas são ótimas para ler códigos, de maneira visual).

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Certo!

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Agora, mesmo que a fonte esteja monoespaçada, os comandos ficam um do lado do outro, diferente de como foi escrito no código HTML. Os comandos estão desorganizados, sem indentação. Para arrumar isso, é possível envelopar toda a tag "code" na tag "pre" 

            
            num = int(input('Digite um número'))
            if num % 2 == 0:
        print(f'O número {num} é PAR')
            else:
        print(f'O número {num} é ÍMPAR')
            print('Fim do programa')


Ao colocar a tag "pre", tudo é considerado, inclusive os espaços, quebras de linha, tabulações, etc. Por exemplo, dentro da tag "pre", a tag "br" não é necessária
Dica: TAB+Shift você retira os espaços e tabulações, e com TAB recoloca denovo. Possível utilizar no código acima
