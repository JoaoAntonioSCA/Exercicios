Alguns tipos de formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag "b". Porém essa tag é não semântica.

Nesta frase temos um termo em destaque feito usando a tag "strong". Que é semântica.

Não há diferença visual entre as duas, pois as duas entregam forma, mas apenas a tag "strong" entrga sentido ao código

Itálico / Ênfase

Nesta frase há um termo em itálcio, feito usando a tag "i". não semântica

Nesta frase há outro termo em itálico, feito usando a tag "em". Semântica

Não há diferença visual entre as duas, pois as duas entregam forma, mas apenas a tag "em" entrga sentido ao código.
As tags "strong" e "em" são novas do HTML5, já as "b" e "i" são do HTML4, ou seja, em um futuro elas tendem a ser consideradas obsoletas.


Texto Marcado

É possível também criar textos marcados usando a tag "mark"

Texto grande e pequeno

É possível criar trechos grandes, com a tag "big". E trechos pequenos, com a tag "small".
Mas "big" é considerado uma tag obsoleta, e "small" não, por que?
Talves porque, ao definir um trecho com a tag "big", você não está dando significado à tag, apenas forma, fazendo ela ficar maior. Já o "small", além de alterar a forma do texto, da um sentido ao trecho pois, por exemplo, em contratos existem as letras pequenas que ficam em baixo dele, que servem para algo, então existe um sentido.
"big" ainda funciona, mas é considerado obsoleto, se você quer alterar o tamanho do texto, é com as CSS.

Texto deletado

É possível marcar um texto como excluído para indicar que ele deve ser lido, mas desconsiderado.

Texto inserido

É possível marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
Existia a tag "u" no HTML4, que fazia a mesma forma, mas não dava significado, ou seja, não semântica. texto como inserido

Texto sobrescrito

Para inserir coisas do tipo x2+3. Com a tag "sup"

Texto subscrito

Para inserir coisas do tipo H2O. Com a tag "sub"
